## What is LXR Hitmarker?

LXR Hitmarker is a professional crosshair and hitmarker disabling system for RedM that enhances immersion in hardcore roleplay environments by removing arcade-style visual feedback during combat.

### Key Features

- **Crosshair Disabling**: Removes the red targeting crosshair when aiming at NPCs and animals
- **Hitmarker Disabling**: Removes visual feedback when shooting or killing NPCs
- **Multi-Framework Support**: Works with LXR-Core, RSG-Core, VORP, and other frameworks
- **Zero Performance Impact**: Optimized native calls with minimal overhead
- **Custom GFX**: Uses custom GFX file for clean targeting experience
- **Immersive Experience**: Perfect for serious roleplay servers

### What Gets Disabled?

1. **Red Crosshair** - The red targeting indicator that appears when aiming at entities
2. **Hitmarker Visual** - The visual feedback shown when bullets hit targets
3. **Kill Marker** - The special confirmation marker shown when killing entities

### Why Use This?

In serious hardcore roleplay environments, arcade-style UI elements break immersion. This resource removes those elements to create a more realistic, immersive combat experience where players must rely on their skills and observation rather than visual assists.

### Who Is This For?

- **Serious RP Servers**: Servers focused on hardcore, realistic roleplay
- **Immersion-Focused Communities**: Communities that value atmosphere and realism
- **PvE-Heavy Servers**: Servers where NPC combat should feel different from PvP
- **Hunting-Focused Servers**: Servers where hunting should be challenging and realistic

## Quick Start

1. **Download** the resource
2. **Place** in your `resources` folder as `lxr-hitmarker`
3. **Add** to `server.cfg`: `ensure lxr-hitmarker`
4. **Configure** in `config.lua` (optional)
5. **Restart** server

That's it! The resource will auto-detect your framework and start working immediately.
