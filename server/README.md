## Server-Side Scripts

This directory contains all server-side Lua scripts for the LXR Hitmarker system.

### Files

- **main.lua** - Server-side validation, logging, and monitoring

### Purpose

Server scripts handle:
- Framework validation
- Resource lifecycle events
- Version checking (future)
- Admin commands
- Server-side logging

### Note

This resource is **primarily client-side**. Server scripts provide:
- Startup validation
- Framework compatibility checks
- Monitoring and logging
- Future expansion capabilities

### Security

No sensitive operations are performed server-side. This resource:
- Does not handle economy
- Does not manipulate player data
- Does not process client requests
- Has zero exploitable vectors

See [Security Guide](../docs/security.md) for details.
